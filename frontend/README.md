# ProofIQ - Frontend

A responsive LMS platform for internal company training, built with:
- Vite + React
- React Router
- Redux Toolkit
- Axios
- TailwindCSS

## Folder Structure
- `src/components/` – shared and role-based UI components
- `src/pages/` – all main page screens, grouped by section
- `src/redux/` – Redux store and slices
- `src/services/` – Axios and API service logic
- `src/hooks/` – app custom hooks
- `src/utils/` – utility functions/constants
- `src/styles/` – project CSS

## Getting Started

```
npm install
npm run dev
```

---

This project mirrors a robust backend (Node/Express/MongoDB, see `../backend`), and features responsive layouts, role-based routing, real file uploads, live leaderboard, and dynamic certificate downloads.
